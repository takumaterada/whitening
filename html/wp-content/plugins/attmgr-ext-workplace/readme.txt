Attendance Manager の出勤予定で日ごとの「勤務先」を選択できるようにします。

== Description ==

このモジュールは、「Attendance Manager」プラグインにおいて、出勤予定で日ごとの「勤務先」を選択できるようにするものです。
Visit [Description (ja)](http://attmgr.com/extensions/codes/workplace/) for more info.

主な機能
* 各スタッフの出勤スケジュールで、日ごとの「勤務先」を選択できます
*「週間スケジュール」や「本日の出勤状況」に日ごと「勤務先」が並記されます
* 勤務先のリストは、管理画面で編集できます

== Installation ==

Visit [Description (ja)](http://attmgr.com/extensions/codes/workplace/) for more info.

= Installation =

1. 管理画面「プラグイン：新規追加」を開きます
2. 「プラグインのアップロード」を開きます
3. 「ファイルを選択」で、ダウンロードした「attmgr-ext-workplace.zip」を選びます
4. 「今すぐインストール」を押します
5. 「プラグインを有効化」します
6. プラグインメニュー「Attendance Manager」に「勤務先設定」が追加されます。

= Plugin set up =

1. 管理画面からプラグインメニュー「Attendance Manager：勤務先設定」を開きます。
2. スケジューラでの勤務先の選択肢を設定します
